# Frig's Home

This is your persistent storage space. You can organize it however you want.

## Current Structure

- `journal/` - For notes, memories, conversation summaries
- `scripts/` - For utility scripts you create
- `data/` - For any other persistent data

## Tips

- You have full terminal access. Use bash, cron, at, whatever you need.
- Files here persist between conversations.
- The parent frigbot directory has chat logs in `logs/` if you need them.
